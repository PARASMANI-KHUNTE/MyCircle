# System Documentation

## System Overview
The platform enabling users to post short-term jobs, offer services, sell items, or rent items within a local area.

## Technology Stack
- **Frontend**: MERN (React.js)
- **Mobile**: React Native
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Images**: Cloudinary
- **Auth**: Google Authentication (OAuth 2.0)
- **AI**: Gemini API

## Architecture
Modular monolith architecture with distinct modules for Auth, Posts, Contact, Safety, and AI.
